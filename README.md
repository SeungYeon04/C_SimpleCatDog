## 🖥 프로젝트 소개 

스파르타 코딩클럽 처음 시작했을 때  
1일 1게임 개발로 만들었던  
다가오는 배고픈 고양이들에게서  
생선가게를 지키기 위해  
사료를 발사해서 만족 시켜 보내는  
고양이vs강아지 게임입니다.
